# Hytalehosting.ai

A modern, gaming-themed website for Hytale game server hosting, built with Next.js 14 and Tailwind CSS.

## Features

- **Responsive Design** - Fully responsive across all devices
- **Dark Gaming Theme** - Deep blues and purples with cyan accents inspired by Hytale's aesthetic
- **Smooth Animations** - Powered by Framer Motion for delightful interactions
- **Modern Stack** - Next.js 14 with App Router, TypeScript, and Tailwind CSS

## Sections

1. **Navigation** - Fixed header with logo and navigation links
2. **Hero Section** - Main heading, feature list, CTA button, and newsletter signup
3. **Pricing Plans** - 5 themed pricing tiers (Kweebec, Tessa, Kyros, Gaia, Varyn)
4. **Server Locations** - Interactive world map with 9 global data center locations
5. **Features Grid** - 6 key hosting features with icons and descriptions
6. **Support Team** - Team member cards showcasing the support staff
7. **Footer** - Multi-column footer with links and social media

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Customization

### Colors

The color palette is defined in `tailwind.config.ts`. Key colors include:

- `accent-cyan`: #00d4ff (primary accent)
- `accent-purple`: #a855f7 (secondary accent)
- `dark-900`: #0a0a1a (background)

### Fonts

- **Display Font:** Orbitron (gaming-style headings)
- **Body Font:** Inter (clean readability)

## Deployment

This project is optimized for deployment on Vercel:

```bash
npm run build
```
